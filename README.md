# CC Counter - عداد الكومبلكس شراكة

تطبيق موبايل لحساب نقاط لعبة الورق "كومبلكس كومبلكس" (الشراكة)

## المميزات

- حساب النقاط تلقائياً
- دعم التدبيل (مضاعفة النقاط)
- سجل الجولات
- واجهة عربية
- أسماء فرق قابلة للتعديل

## قواعد اللعبة

| البطاقة              | النقاط | مدبلة |
| -------------------- | ------ | ----- |
| الأكلة (4 ورقات)     | -15    | -     |
| ورقة ديناري          | -10    | -     |
| بنت (Q)              | -25    | -50   |
| شيخ القبة (K hearts) | -75    | -150  |

مجموع كل جولة = -500 نقطة

## التثبيت

```bash
pip install kivy arabic-reshaper python-bidi pillow
```

## التشغيل

```bash
python main.py
```

## بناء APK

استخدم Google Colab مع Buildozer - راجع ملف build_apk.txt
